Classic editor

Before version 1.3, there was a seperate "Edit" page instead of the current Home page editing, which is more reliable and doesn't require switching tabs.

Why was it removed?

The updated Home page allows immediate editing and previewing how album covers will look, all quick and simple, and no need to go back and forth between the two pages

Use the Classic Editor

The seperate Editor page will always be available at audomark.sipped.org/editor for those who want to use it.
